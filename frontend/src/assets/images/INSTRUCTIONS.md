# 📸 Image Assets Instructions

## Quick Setup

### 1. Add DMU Logo
Place your DMU logo PNG file here:
```
frontend/src/assets/images/branding/dmu-logo.png
```

**Requirements**:
- Format: PNG with transparent background
- Size: 512x512px recommended
- File size: < 200KB

### 2. Add Campus Images (Optional)
Place campus photos here:
```
frontend/src/assets/images/campus/campus-main.jpg
frontend/src/assets/images/campus/library.jpg
```

**Requirements**:
- Format: JPG
- Size: 1920x1080px recommended
- File size: < 500KB each

### 3. Restart Server
```bash
cd frontend
npm run dev
```

### 4. Verify
- Login page: http://localhost:5173/login
- Landing page: http://localhost:5173/
